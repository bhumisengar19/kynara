# 🌌 Kynara.ai — AI English Coach & Conversational Platform

Kynara.ai is a **full-stack AI-powered application** that combines a modern chat system with a **real-time English speaking coach**, delivering a premium and interactive learning experience.

> 🚀 Built as a **product-level project** — not just a chatbot.

---

## 🎯 Key Highlights

* 🧠 AI Chat + English Coach in one platform
* 🎤 Voice-based interaction
* 🎮 Gamified learning (XP, streaks, levels)
* 🎭 Roleplay-based practice
* 📊 Learning analytics & feedback
* 🎨 Premium UI with animations

---

# ⚙️ Tech Stack (Feature-wise Breakdown)

This section clearly explains **which technology is used for which feature** 👇

---

## 💬 1. AI Chat System

**Features:**

* Conversational AI
* Multiple response modes
* Chat history

**Tech Used:**

* `React + Vite` → UI rendering
* `Node.js + Express` → API handling
* `MongoDB + Mongoose` → Chat storage
* `Google Gemini API` → AI responses

---

## 🗣 2. English Practice Mode (Core Feature)

**Features:**

* Grammar correction
* AI tutor responses
* Learning notes

**Tech Used:**

* `Gemini API` → Grammar + response generation
* `Custom Prompt Engineering` → Combined correction + explanation
* `React State / Context` → Real-time UI updates

---

## 🎤 3. Voice Interaction

**Features:**

* Speech-to-Text (User speaking)
* Text-to-Speech (AI speaking)

**Tech Used:**

* `Web Speech API` → Speech recognition (FREE)
* `SpeechSynthesis API` → Voice output
* `JavaScript Audio APIs` → Voice control

---

## 🎭 4. Roleplay System

**Features:**

* Scenario-based conversation
* Dynamic follow-ups

**Tech Used:**

* `Predefined Question Bank (JSON)`
* `Gemini API` → Context-aware responses
* `State Management` → Mode switching

---

## 🎮 5. Gamification System

**Features:**

* 🔥 Streak tracking
* ⚡ XP system
* 🏆 Levels
* 🎯 Daily goals

**Tech Used:**

* `localStorage` → Persist streak + XP
* `React State` → Real-time updates
* `Custom Logic (JS)` → XP + level calculation

---

## 🧾 6. Learning Notes System

**Features:**

* Mistake tracking
* Correction history

**Tech Used:**

* `React State` → Store session notes
* `localStorage / MongoDB` → Persistence
* `Gemini API` → Generate corrections

---

## 📊 7. Analytics & Feedback

**Features:**

* Accuracy tracking
* Practice stats
* Improvement insights

**Tech Used:**

* `Custom JS Logic` → Score calculation
* `React Components` → UI display
* `State Management` → Data tracking

---

## 📂 8. File Upload & Analysis

**Features:**

* Upload PDFs, DOCX, CSV
* AI reads files

**Tech Used:**

* `Multer` → File uploads
* `pdf-parse` → PDF reading
* `mammoth` → DOCX parsing
* `csv-parser` → CSV parsing
* `Gemini API` → Content understanding

---

## 🎨 9. UI / UX System

**Features:**

* Glassmorphism UI
* Animations
* Dark/Light theme

**Tech Used:**

* `Tailwind CSS` → Styling
* `Framer Motion` → Animations
* `Lucide React` → Icons

---

## 🌌 10. Advanced UI Effects

**Features:**

* Background animations
* Immersive design

**Tech Used:**

* `Three.js / React Three Fiber` → Visual effects

---

## 🔐 11. Authentication System

**Features:**

* User login/session

**Tech Used:**

* `JWT (JSON Web Token)` → Auth
* `Express Middleware` → Protected routes

# 🚀 Getting Started

### 1. Clone the repo

```
git clone https://github.com/bhumisengar19/kynara.git
```

### 2. Install dependencies

```
cd frontend
npm install

cd ../backend
npm install
```

### 3. Run project

```
# frontend
npm run dev

# backend
npm start
```
